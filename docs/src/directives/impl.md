# `.impl`

This directive computes boolean implication, that is, given two inputs whether one implies the other by the following truth table.

| \\( a \\)   | \\( b \\)   | \\( a \Rightarrow b \\)   |
| ----------- | ----------- | ------------------------- |
| false       | false       | true                      |
| false       | true        | false                     |
| true        | false       | true                      |
| true        | true        | true                      |
