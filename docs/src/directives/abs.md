# .abs

Computes the absolute value or magnitude of a given number, that is, the unsigned distance between it and zero.

\\( \mathtt{.abs}(x) = \vert{x}\vert = \begin{cases} x & \text{if}\ x \geq 0 \\\\ -x & \text{if}\ x < 0 \end{cases} \\)

## Interface

- **Inputs:**
	- A real number _x_
- **Output:**
	- |_x_|
